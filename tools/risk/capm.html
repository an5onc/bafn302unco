CAPM Required Return

💡
Why This Matters

The Capital Asset Pricing Model (CAPM) tells you the minimum required return an investor should demand given a stock's systematic risk (β). If expected return < CAPM required return, the stock is overvalued.

Formula:
  • rᵢ = rf + β(rₘ − rf)
  • rf — risk-free rate (e.g. T-bill rate)
  • β — beta (systematic risk)
  • rₘ — expected market return (or enter MRP directly)
  • rₘ − rf — market risk premium (MRP)
What You Know

If both rₘ and MRP are provided, rₘ takes priority.
