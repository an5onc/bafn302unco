Portfolio Return

Why This Matters

Portfolio return is the weighted average of individual asset returns, where weights represent each asset's share of the total portfolio value.

Formula:
  • rₚ = Σ (wᵢ × rᵢ)
  • Weights must sum to 1.0 (100%). Enter as decimals (e.g. 0.40 for 40%).
