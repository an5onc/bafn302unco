Formula

Monthly Payment:
PMT = P × [r(1+r)^n] / [(1+r)^n - 1]

Where:
• P = Principal loan amount
• r = Period interest rate (annual rate / periods per year)
• n = Total number of payments

HP 10bII+ Calculator Steps

  1. Clear: SHIFT C ALL
  2. Set periods per year: 12 SHIFT P/YR
  3. Enter loan amount: [amount] PV
  4. Enter interest rate: [rate] I/YR
  5. Enter number of payments: [years × 12] N
  6. Calculate payment: PMT (result will be negative)
  7. For amortization: SHIFT AMORT
