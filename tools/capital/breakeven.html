Break-Even Analysis

💡
Why This Matters

Break-even analysis tells you how many units (or dollars of sales) you need to cover all costs. Below break-even → losses. Above → profits. Operating leverage determines how fast profits grow once you clear the break-even point.

Key Formulas:
  • Q_BE = FC / (P − VC)   units
  • S_BE = FC / CM Ratio   dollars
  • CM Ratio = (P − VC) / P
  • DOL = Q(P−VC) / [Q(P−VC) − FC]
  • % ΔProfit ≈ DOL × % ΔSales
